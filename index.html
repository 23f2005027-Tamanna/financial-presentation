Calculating Returns


import numpy as np

def calculate_log_returns(prices):
    """
    Compute log returns from price data
    """
    # Use numpy for vectorization
    returns = np.diff(np.log(prices))
    return returns

Black-Scholes Formula

For call option pricing:

\[ C(S_t, t) = N(d_1)S_t - N(d_2)Ke^{-r(T-t)} \]

Where:

\[ d_1 = \frac{ln(\frac{S_t}{K}) + (r + \frac{\sigma^2}{2})(T-t)}{\sigma\sqrt{T-t}} \]
This is the standard Black-Scholes formula used for European Call Options. N(.) represents the cumulative distribution function.
